# 内存分配

## 实现方法

非常原始的方法。

将10MB以上的内存划分为用户进程使用。这些用户进程是除开tty、fs等进程外由操作系统创建的进程。这些进程使用的内存空间的大小是1MB。

给进程分配内存空间，就是计算出一个进程的内存空间的初始地址。

进程的内存空间的初始地址 = (进程的pid - (操作系统预置的进程数量))*每个进程的内存空间的大小 + 10。
